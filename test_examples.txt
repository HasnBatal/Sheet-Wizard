========================================
CUSTOM FUNCTION TEST EXAMPLES
========================================

Use these examples to test the new custom function feature with sample_data.csv

========================================
EXAMPLE 1: Extract Email Domain
========================================
Column: Email
Custom Function:
function customTransform($value) {
    $parts = explode('@', $value);
    return isset($parts[1]) ? $parts[1] : $value;
}

Expected Result: example.com

========================================
EXAMPLE 2: Format Phone Numbers
========================================
Column: Phone
Custom Function:
function customTransform($value) {
    $clean = preg_replace('/[^0-9]/', '', $value);
    if (strlen($clean) === 7) {
        return preg_replace('/^(\d{3})(\d{4})$/', '$1-$2', $clean);
    }
    return $value;
}

Expected Result: 555-1234

========================================
EXAMPLE 4: Name to Initials
========================================
Column: Name
Custom Function:
function customTransform($value) {
    $parts = explode(' ', trim($value));
    $initials = '';
    foreach ($parts as $part) {
        $initials .= strtoupper(substr($part, 0, 1));
    }
    return $initials;
}

Expected Result: JD (for John Doe)

========================================
EXAMPLE 9: Generate Username from Email
========================================
Column: Email
Custom Function:
function customTransform($value) {
    $parts = explode('@', $value);
    $username = $parts[0];
    return strtolower(str_replace('.', '_', $username));
}

Expected Result: john (from john@example.com)

========================================
EXAMPLE 10: Conditional Formatting
========================================
Column: City
Custom Function:
function customTransform($value) {
    $length = strlen($value);
    if ($length > 10) {
        return strtoupper($value) . ' (LONG)';
    }
    return $value;
}

Expected Result: PHILADELPHIA (LONG)

========================================

}
Expected: Error message about function name

Test 2 - Syntax Error:
function customTransform($value) {

}
Expected: Syntax error message (missing semicolon)

========================================
ADVANCED COMBINATIONS
========================================

Example: Complete Email Processing
Column: Email
Options: ✓ Trim Spaces, ✓ Remove Duplicates
Pipeline: strtolower
Custom Function:
function customTransform($value) {
    $parts = explode('@', $value);
    return isset($parts[1]) ? $parts[1] : $value;
}

This will:
1. Trim spaces
2. Convert to lowercase
3. Extract domain
4. Remove duplicate domains
